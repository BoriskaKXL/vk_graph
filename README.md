## VK Audience Graph

Граф, визуализирующий схожесть аудитории крупнейших пабликов ВК

### Технологический стек
- Python + Numpy для анализа данных
- Vue.js на фронтенде

### Quickstart
1. Установить node.js 14

2. Установить зависимости:
  ```bash
  cd ./web
  npm install
  ```
3. Запустить dev-сервер
  ```bash
  npm run serve
  ```
